PROD 6 CHANGELOG
===============

2013-08-20 Release 1.4.1

	* Added new WebRemote interface. Currently supported by the following internet browsers: Firefox, Google Chrome.
	* Improved the usability of text entry on the front panel.
	* Improved screen lock feature. If enabled, the screen will go to screenasver straight away after boot.
	* Adding an on-screen message to indicate when the unit is under a remote control.
	* Changed power step size to implement 100 steps.

	* Ethernet LEDs have been enabled.
	* Updated web remote. Now only password is required to login.
	* More accurate Peak Deviation reading.
	* Improved encoder acceleration performance.
	* Fixed swapped Mono L/Mono R audio modes.
	* Fixing web remote reliability.
